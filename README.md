## 手里有糖个人博客简介

### 必备环境

- 安装 `Node.js`（version：8.11.0+）
- 确保 `npm` 或者 `yarn` 可用

### 线上地址：

[手里有糖个人博客]([http://sugar.whgjh.top](http://sugar.whgjh.top/))

### 技术点

- React hoocks
- nodejs
- jwt
- mysql(Sequelize)



## 下载工程代码

```javascript
git clone https://github.com/wuliaodexuanze/personal-blog-react.git
```

### 安装依赖包

``` javascript
cd personal-blog-react && npm install
```

### 运行

``` javascript
npm run start
```
